# Middleware

A layer wraps every route beneath it. Add tracing, authentication, compression,
or timeouts by composing layers on the router.

```rust
let app = Router::new()
    .route(show)
    .layer(Trace::new())
    .layer(Compression::gzip())
    .layer(Timeout::secs(10));
```

## Writing a layer

A layer is a function from a handler to a handler. It can run code before the
inner handler, after it, or short-circuit entirely:

```rust
async fn require_auth(req: Request, next: Next) -> Response {
    if req.header("authorization").is_none() {
        return StatusCode::UNAUTHORIZED.into_response();
    }
    next.run(req).await
}
```

## Order is explicit

Layers apply outermost-first, in the order you list them. Because the stack is
assembled at compile time, the request path through your middleware is fixed and
inspectable — no hidden global registry.
