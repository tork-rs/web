# `#[tork::main]`

Marks the asynchronous entry point of your program. It rewrites your `async fn main`
into a normal synchronous `main` that builds a multi-threaded Tokio runtime and runs
your code on it, so you do not set up the runtime yourself.

**Applies to:** an `async fn` (your `main`).

**Options:** none.

```rust
use tork::App;

#[tork::main]
async fn main() -> tork::Result<()> {
    App::new()
        .serve("0.0.0.0:8000")
        .await
}
```

## Rules

- The function must be `async`. Applying it to a non-async function is a compile
  error.
- The return type is kept as you wrote it, so `tork::Result<()>` works as usual.
